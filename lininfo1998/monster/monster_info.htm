Formula of experience of monsters:
- Experience = level x level + 1
- If monster's level is 9 or higher, add bonus 50.
- If monster's AC is 0, 1 or 2, add bonus: (7 - ac).
- If monster's AC is below 0, add bonus: (7 - ac) x 2.
